# **리눅스 top, ps, jobs, kill 명령어

## **top**
-시스템 실시간 모니터링 도구로 CPU, 메모리 사용량, 실행 중인 프로세스를 실시간으로 보여준다.
| 단축키 | 기능 |
|:------:|:-------------------------|
|top     | 실행 |
|P|cpu사용량 기준 정렬|
|M|메모리 사용량 기준 정렬|
|q       | top 종료 |
|k->PID->시그널번호| 프로세스 종료|

## **ps**
-현재 실행 중인 프로세스 목록을 스냅샷 형태로 출력한다.<br>
-PID 확인, 프로세스의 부모/자식 관계 확인 시 사용
|단축키|기능|
|:------:|:------------------------------|
|ps|실행|
|ps -e|모든 프로세스 출력|
|ps -f|자세한 정보 출력|
|ps aux+|프로세스 | 특정 프로세스 찾음|

## **jobs**
-백그라운드에서 실행 중인 작업 목록을 보여준다.
|단축키 | 기능 |
|:-------:|:-----------------------------|
| jobs | 실행|
| jobs -i | PID함꼐 표기 |
| fg %n | n번을 포어그라운드로 가져옴|
| bg %n | n번을 백그라운드에서 재시작|

##  **kill**
-특정 PID를 가진 프로세스에 시그널을 보내 제어한다. (대부분 프로세스 종료로 사용)
|단축키 | 기능 |
|:--------:|:----------------|
| kill [PID] | 종료 요청 |
| kill -9 [PID] | 강제 종료 |
| kill %1 | jobs 번호 1 종료 |

| 시그널 번호 | 기능 |
|:-----------:|:------------------------------|
| SIGTERM 15 | 정상 종료 요청 |
| SIGKILL 9 | 강제 종료 |
| SIGINT 2 | 프로그램 종료 요청(Ctrl + C) |
| SIGSTOP / SIGTSTP | 프로세스 일시 정지 |
| SIGCONT | 일시정지된 프로세스 재시작 |

예시
kill 123 (PID 123 프로세스 종료)<br>
kill %1 (job에서 1번 종료)<br>
⚠️kill -9 123 (강제 종료)<br>
ㄴ(시스템 손상 가능성, kill -15(SIGTERM) 먼저 실행 권장)
